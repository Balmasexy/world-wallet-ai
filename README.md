# World Wallet AI - Monorepo

Full-stack monorepo for the World Wallet AI application.

## Directory Structure

### apps/
- **web/** - React frontend application

### services/
- **api/** - FastAPI backend
- **auth/** - Authentication service
- **wallet/** - Wallet management service
- **ai/** - AI agents and intelligence service

### packages/
- **shared/** - Shared utilities, types, and constants

### infrastructure/
- **docker/** - Docker configurations and images

## Quick Start

```bash
docker-compose up
```
